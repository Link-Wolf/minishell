<!-- GETTING STARTED -->
## Getting Started

Because it's a simple C project, there isn't much to say here

### Prerequisites

Having a C compiler like cc, gcc or clang

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Link-Wolf/minishell.git
   ```
2. Compile the project
   ```sh
   cd minishell; make
   ```
3. Execute it
   ```sh
   ./minishell
   ```
